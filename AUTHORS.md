# Treaty Bridge Project

## Creator
This project was initiated and built by a Mi'kmaw community member
exploring how code can serve language revitalization and cultural
preservation.

## Acknowledgments
- Elder Albert Marshall for the Two-Eyed Seeing (Etuaptmumk) framework
- All Mi'kmaw Elders and knowledge keepers past, present, and future
- The resilience of the Mi'kmaw language against centuries of suppression

## How to Contribute
We welcome:
- Corrections from Mi'kmaw language speakers
- Improvements to data structures
- Documentation enhancements
- Respectful adaptations for other Indigenous communities

Please see CONTRIBUTING.md for guidelines.
